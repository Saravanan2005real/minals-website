## About the Project

Minals (Ramaiah Enterprises) is a legacy brand with 60+ years of manufacturing excellence. This website serves as the company's official digital presence, showcasing their product catalogue, dealer network, and contact information.

Built with **Next.js 15 App Router**, **TypeScript**, and **Tailwind CSS v3**, the site is optimized for speed, SEO, and premium visual impact.

## Recent Updates & Enhancements

### 🚀 Complete 44-Product Catalog
- **Full Range Integration**: Added the complete 44-product catalog across Sanitizers, Cleaning, Wellness, and Food categories.
- **Descriptive Naming**: Renamed all product images to clean, descriptive kebab-case (e.g., `premier-toilet-cleaner.png`) for better SEO and project alignment.
- **Quantity-Based Enquiries**: Enhanced the WhatsApp enquiry logic to include specific quantities for each product.

### 📱 Mobile Excellence
- **Above-the-Fold Optimization**: Redesigned the homepage hero section for mobile to ensure the "Explore Products" CTA is visible immediately without scrolling.
- **Responsive Grids**: Optimized category and product grids for seamless browsing on any device.

### 🏛️ Professional UI Overhaul
- **Dealers Page**: Redesigned with premium card-based features, high-end serif typography (Playfair Display), and precise map alignment.
- **Contact Page**: Streamlined contact details into a professional, easy-to-read vertical layout.
- **Enquiry Form**: Simplified to a high-conversion, single-column format removing redundant fields like "Company Name" and "State".

### 💬 "Inbuilt" WhatsApp Messaging
- **Automated Outreach**: Added professional pre-filled messages to all WhatsApp CTAs, ensuring users can start an enquiry with a single click.
